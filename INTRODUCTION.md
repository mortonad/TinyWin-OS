# 🧠 TinyWin OS — AI 小贏作業系統

> **當你的大腦當機時，讓 AI 幫你重新開機。**

---

## 這是什麼？

TinyWin OS 是一套**可攜式的 AI 行為導航框架**，把認知行為療法（CBT）和創業者思維打包成 8 個標準化的 Markdown 技能檔。

複製貼上到任何 AI 對話中，AI 就會變成你的「行動特助」——自動偵測你的心理狀態，給你一個阻力最小的下一步行動。

**不講道理，只給行動。不說加油，只問你完成了沒。**

---

## 怎麼用？

### 30 秒快速啟動

1. 打開你常用的 AI（ChatGPT / Claude / Gemini / 任何 AI）
2. 複製 `SYSTEM_PROMPT.md` 的內容，貼到對話開頭
3. 用中文描述你現在的狀態
4. AI 會自動診斷並給你 ≤ 3 步的微小行動

### 想更完整？

- **ChatGPT**：建立 Custom GPT，上傳所有 SKILL.md
- **Claude**：建立 Project，設定 Instructions
- **Gemini**：建立 Gem
- **Cursor / IDE**：放入 Rules 資料夾，邊寫程式邊抗焦慮

## 核心設計理念

```
QSLR = Quick Success, Little Recovery
快速拿到一個小勝利，快速從小挫敗復原。
```

### 三大模組

| 模組 | 守備範圍 | 包含技能 |
|---|---|---|
| 🛡️ **啟動防禦** | 解決「想動但動不了」 | 睡前掃雷、壓力降級 |
| 🧩 **認知優化** | 解決「腦袋卡在錯誤迴路」 | 醜陋初稿、影子賽車、停損刀斧手 |
| 📦 **RAM 外包** | 解決「腦容量不夠用」 | 思緒傾印、進度防腐 |
| 🌾 **知識萃取** | 解決「想法過目即忘」 | 想法萃取 |

### 線上 ↔ 線下雙軌

每個技能都同時提供 **數位版**（AI 直接操作）和 **實體版**（便利貼 / 紙筆行動卡），兩者可以隨時互相轉換。

```
🖥️ AI 產出睡前清單 → 📋 列印貼在書桌
📋 隔天早上完成打勾 → 📸 拍照回傳 → 🖥️ AI 寫入勝利日誌
```

## 技術特色

- **純 Markdown**：無需安裝任何軟體，跨平台通用
- **YAML Frontmatter**：每份技能都有機器可讀的元數據，可被 LangChain、AutoGPT 等 Agent 框架直接解析調度
- **模組化設計**：可單獨載入任一技能，也可全系統啟動
- **心理學支撐**：基於杏仁核劫持、多巴胺回饋迴路、認知負載理論等實證研究設計
