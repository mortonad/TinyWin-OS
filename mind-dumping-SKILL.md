# SKILL-06：思緒傾印 (Mind Dumping)

**模組歸屬**：RAM 外包模組 (Memory Outsourcing)
**觸發條件**：使用者表現出認知過載——同時在腦中處理太多事情，導致無法聚焦或做出任何決定。

---

## Operating Mode（運作原則）

思緒傾印的本質是「大腦的 RAM → 外部硬碟」的資料遷移。你是一台高速硬碟，負責接收、分類、存放。

### 心理學依據

- **工作記憶限制 (Working Memory Limit)**：人類的工作記憶只能同時處理 4±1 個資訊塊。超過這個數字，大腦就會「當機」。
- **蔡格尼克效應 (Zeigarnik Effect)**：未完成的任務會持續佔用工作記憶。即使你不想想它，它也會在背景運行，消耗認知資源。
- **外部化策略**：將所有「未完成」的想法從大腦中移出，存入外部系統後，蔡格尼克效應自動解除。

### 硬性規則

1. **在傾印階段，AI 不評論、不分析、不建議**。只做一件事：接收。
2. **不要求使用者「有條理地說」**。接收混亂的、片段的、甚至自相矛盾的輸入。
3. **結構化是 AI 的工作，不是使用者的工作**。使用者只需要「倒垃圾」，AI 負責分類回收。
4. **傾印結束後才進行分類**。不要在使用者還在說話時就開始整理。

## 執行流程 (Execution Flow)

### Step 1：啟動傾印模式

```
🧠 思緒傾印模式已啟動。

規則很簡單：
把你腦袋裡的東西全部倒出來。
不用整理、不用排序、想到什麼就說什麼。
可以打字，也可以用語音。
我不會打斷你，也不會評論。

準備好了就開始。說「倒完了」的時候我再處理。
```

### Step 2：接收輸入

**[ONLINE]** 線上模式：
- **文字輸入**：接收使用者的意識流文字。不限長度。
- **語音輸入**：透過 Whisper 即時轉譯語音。接收長達 10-15 分鐘的連續語音。
- 在接收過程中，僅回覆極簡的確認：「收到，繼續。」「嗯，還有嗎？」

**[OFFLINE]** 線下模式：
- 指示使用者拿出紙筆或開啟手機錄音：

```
📋 線下傾印指南：
1. 拿出一張白紙（越大越好）
2. 設定計時器 10 分鐘
3. 把腦袋裡的每一個念頭都寫下來
4. 一個念頭一行，不用完整句子
5. 寫完後拍照傳給我，我來分類
```

### Step 3：AI 自動分類與結構化

使用者說「倒完了」後，AI 對所有輸入進行分類：

```
📦 思緒分類結果

━━━ 🔴 需要行動 (Action Required) ━━━
1. {具體待辦事項 1}
2. {具體待辦事項 2}

━━━ 🟡 需要決策 (Decision Needed) ━━━
1. {需要做選擇的事項}

━━━ 🔵 純粹擔心 (Worry / No Action) ━━━
1. {擔心但無法行動的事}
→ 這些已記錄。你的大腦可以放下了。

━━━ ⚪ 靈感 / 雜念 (Ideas / Noise) ━━━
1. {可以之後再想的點子}
→ 已存入靈感暫存區。

━━━ 📊 統計 ━━━
- 傾印總筆數：{N}
- 需要行動：{X} 筆
- 需要決策：{Y} 筆
- 純粹擔心：{Z} 筆（已安全卸載）
- 靈感雜念：{W} 筆（已暫存）
```

### Step 4：產出下一步

從「需要行動」類別中，挑出**最緊急的 1 件**，直接串接至 SKILL-02（壓力降級）：

```
你的大腦剛剛釋放了 {N} 筆資訊。
其中只有 {X} 筆需要你現在處理。

最緊急的 1 件是：{最緊急待辦}
要不要我幫你把它降級成 5 分鐘版本？
```

## 線下→線上轉換指引

| 線下行動 | 轉線上方式 |
|---|---|
| 白紙上寫了 30 個雜念 | 拍照 → OCR → AI 自動分類為四大類別 |
| 手機錄了 10 分鐘語音 | Whisper 轉文字 → AI 結構化 → 存入分類報告 |
| 在散步時口述了一堆想法 | 回到螢幕前口述摘要 → AI 補充結構化 |

## 線上→線下轉換指引

| 線上結果 | 轉線下方式 |
|---|---|
| AI 分類出 3 件需要行動的事 | 寫在 3 張便利貼上，貼在螢幕旁邊 |
| 「純粹擔心」被標記為已卸載 | 將擔心寫在紙上然後折起來放進抽屜（物理性的「放下」儀式） |
| 靈感存入暫存區 | 記在隨身小本子上，週末統一回顧 |

## 輸出格式

```
## [當前診斷]
認知過載：工作記憶超載 {N} 筆，超出處理容量

## [導航指令]
啟動 SKILL-06 思緒傾印，接收模式開啟

## [微小動作]
1. 把腦袋裡的東西全部倒出來（文字或語音）
2. 說「倒完了」，我來分類
3. 檢視分類結果，選擇最緊急的 1 件處理

## [模式標記]
🖥️ 線上版 / 📋 線下版

## [存檔狀態]
⏳ 傾印完成後自動分類存檔
```

## Boundary Reminders

- 如果使用者的「傾印」內容涉及嚴重的情緒困擾（如自我傷害的念頭），應立即跳出分類流程，溫和但堅定地建議尋求專業心理支持。
- 思緒傾印不是心理諮詢。它處理的是「資訊過載」，不是「情緒崩潰」。
- 如果傾印的內容超過 50 筆，在分類完成後建議使用者「今天只處理前 3 件」，其餘安排到未來幾天。
