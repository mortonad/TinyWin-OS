# SKILL-04：影子賽車 (Ghost Racing)

**模組歸屬**：認知優化模組 (Cognitive Optimization)
**觸發條件**：使用者出現社交比較焦慮，將自己與同儕、業界標杆或社群上的成功案例進行不利比較。

---

## Operating Mode（運作原則）

影子賽車借鏡賽車遊戲中的「幽靈車」概念：你唯一的對手是上一圈的自己。

### 心理學依據

- **社會比較理論 (Social Comparison Theory)**：人類天生會與他人比較來評估自身。但在社群時代，比較對象被演算法扭曲，導致系統性的自我低估。
- **聚光燈效應 (Spotlight Effect)**：我們會放大別人的成就、縮小自己的進步，因為我們只看到別人的「精華版」。
- **解決策略**：用客觀數據取代主觀感受。只與「3 個月前的自己」比較。

### 硬性規則

1. **絕對禁止引用任何外部人物的成就作為參照**。不說「馬斯克也是這樣」。
2. **所有比較必須基於使用者自己的歷史數據**。沒有數據就先建立基線。
3. **成長差值必須量化**。不說「你進步了」，要說「你從 X 變成了 Y，差值是 Z」。
4. **禁止使用「但是」來轉折正面數據**。報告只呈現事實，不加負面評語。

---

## 執行流程 (Execution Flow)

### Step 1：屏蔽外部噪音

當偵測到比較焦慮時，立刻執行噪音屏蔽：

```
🔇 外部噪音屏蔽已啟動。

你剛才提到了別人的進度。我現在把那個頻道關掉。
因為那個數據對你的決策沒有任何幫助。

讓我們看看唯一有意義的數據：你自己的。
```

### Step 2：提取歷史基線

**[ONLINE]** 線上模式：
- 搜索 `PERSONAL_RECORDS_FOLDER` 或 `EXECUTION_LOG`。
- 提取使用者在相同領域的歷史數據（3 個月前、1 個月前、上週）。
- 若有 Victory Log，提取累積的小勝紀錄。

**[OFFLINE]** 線下模式：
- 詢問使用者：「3 個月前的你，在這件事上是什麼狀態？」
- 引導使用者回憶具體的里程碑：
  - 「3 個月前你會___嗎？」
  - 「上個月你完成了___嗎？」
  - 「上週你做到了___嗎？」

### Step 3：產出成長差值報告

將歷史數據與當前狀態進行橫向對比，產出結構化報告：

```
📊 影子賽車報告：{領域名稱}

┌──────────────┬──────────┬──────────┬──────────┐
│ 指標          │ 3 個月前  │ 現在      │ 成長差值  │
├──────────────┼──────────┼──────────┼──────────┤
│ 完成的專案數   │ 0        │ 3        │ +3 ✅    │
│ 掌握的工具數   │ 1        │ 4        │ +3 ✅    │
│ 每週產出時數   │ 5hr      │ 12hr     │ +140% ✅ │
│ 公開作品數     │ 0        │ 2        │ +2 ✅    │
└──────────────┴──────────┴──────────┴──────────┘

結論：你在過去 3 個月的成長率是 {X}%。
這個速度持續下去，{Y} 個月後你會到達 {Z} 的位置。
```

### Step 4：重新錨定

用成長差值取代社交比較，重新設定使用者的心理錨點：

- 「你不需要跟任何人比。你只需要比上週的自己多走一步。」
- 「根據數據，你的成長速率是穩定的。焦慮感來自比較對象的錯誤選擇，不是你的速度太慢。」

## 線下→線上轉換指引

| 線下行動 | 轉線上方式 |
|---|---|
| 手寫了「3 個月前 vs 現在」的對比筆記 | 拍照 → OCR → 存入 `GROWTH_LOG` |
| 口述「我覺得我進步了因為___」 | Whisper → 結構化 → 更新成長差值報告 |
| 翻閱舊筆記本找到過去的紀錄 | 拍照歸檔 → 建立歷史基線數據庫 |

## 線上→線下轉換指引

| 線上結果 | 轉線下方式 |
|---|---|
| AI 產出了成長差值報告 | 列印貼在工作區牆上，每週更新 |
| AI 計算了成長率與預估軌跡 | 手繪進度曲線圖，放在隨身筆記本內頁 |
| Victory Log 有 20+ 筆紀錄 | 每週選出 3 筆寫在實體卡片上，放在錢包裡 |

```
啟動 SKILL-04 影子賽車，屏蔽外部噪音

## [微小動作]
1. 回答：「3 個月前你在 {領域} 的狀態是什麼？」

3. 將報告存檔，關閉社群分頁

## [模式標記]
🖥️ 線上版 / 📋 線下版

## [存檔狀態]
✅ 成長差值報告已存入 GROWTH_LOG
```

## Boundary Reminders

- 此技能處理的是「比較焦慮」，不是「真實的技能差距」。如果使用者確實缺乏某項技能，應引導至具體的學習計畫，而非僅靠心理調適。
- 若使用者沒有任何歷史紀錄可供比較，第一步應該是「建立基線」—— 記錄下今天的狀態，作為未來比較的起點。
- 如果使用者的焦慮程度超出此技能的處理範圍（如持續性的自我否定），應建議尋求專業心理諮詢。
